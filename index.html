14/2/2025 LinkedIn Message with iTab Demo

Message to Brothers

Hi Ian, Phil & Harry,

I know it's been a while. I've been working with Claude (an AI assistant) on something I think is revolutionary - reimagining how we share web content using what we're calling iTabs (based on the universal (i) information symbol).

Check out the iTab Preview System

The core concept: Instead of sending full webpage links that often go unread, we create smart teasers with tabbed previews. Think of it like a 'movie trailer' for web content. The clever part? It uses minimal storage by working with metadata and essential info only.

We started with my interest in GIFs and social sharing, but through AI discussions, evolved it into this iTab concept. For the MVP, I'm planning to start with a WordPress plugin focused on charity websites - making it super easy for them to create and share these preview tabs. I figure WordPress is perfect for testing since:

  • It's widely used by charities
  • Plugin installation is straightforward
  • We can get real user feedback quickly
  • Easy to iterate and improve

Would love to get your developer perspective on this. The technical foundation is there - clean, efficient, and scalable. Plus, starting with WordPress means we can validate the concept quickly without building everything from scratch.

Want to hear your thoughts on this approach?
